# Contributing to Android SMS Gateway

First off, thanks for taking the time to contribute! ❤️

We love your help and want to make the project as good as possible. Here are a few guidelines for contributing:

## Reporting Bugs
- Check if the bug is already reported in the Issues.
- If not, create a new issue using the **Bug Report** template.
- Please provide as much detail as possible (Android version, device model, server logs).

## Feature Requests
- Open an issue using the **Feature Request** template.
- Explain why the feature would be useful and how it should work.

## Pull Requests
1. Fork the repository.
2. Create a new branch for your change.
3. Keep your PRs focused and concise.
4. If you've added code that should be tested, add tests.
5. Submit the PR and wait for review!
